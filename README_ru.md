# P2P Обучение на Pion WebRTC 🎓

**P2P Обучение** — это веб-приложение для прямого peer-to-peer взаимодействия между учителем и учеником, построенное на базе Pion WebRTC и Golang. Приложение позволяет организовать видео-конференцию с чатом, обменом файлами и эксклюзивными возможностями для учителя.

## ✨ Возможности

### 🎥 Видеосвязь
- Прямое P2P соединение (WebRTC)
- Камера собеседника крупным планом, своё видео — компактно
- Индикаторы работы камеры и микрофона у обоих участников
- Демонстрация экрана для учителя

### 💬 Чат
- Удобный сворачиваемый чат (вертикальная полоса)
- Обмен текстовыми сообщениями в реальном времени
- Отправка файлов
- Автоматическое уведомление о включении/выключении устройств

### 👨‍🏫 Функции для учителя
- Кнопка **"Внимание!"** — при нажатии у ученика:
  - Чат автоматически разворачивается
  - Появляется яркое сообщение ⚠️ ВНИМАНИЕ ⚠️
- Управление своей камерой, микрофоном и демонстрацией экрана
- Учитель остаётся в сессии даже после отключения ученика

### 🧑‍🎓 Функции для ученика
- Управление своей камерой и микрофоном
- Индикатор ожидания учителя (если учитель ещё не подключился)
- Автоматическое подключение при появлении учителя

### 🛠 Технические особенности
- **Умное ожидание**: индикатор показывается только если учитель действительно не в сети
- **Надёжное переподключение**: при обрыве соединения клиент автоматически переподключается
- **Статус-бар**: отображение состояния WebSocket, P2P и медиа-устройств
- **Панель отладки**: скрываемая под спойлер, содержит подробную информацию о работе приложения

## 📋 Требования

- Go 1.16 или выше
- Современный браузер с поддержкой WebRTC (Chrome, Firefox, Safari, Edge)
- Доступ к камере и микрофону
- Для работы по HTTPS — самоподписанные или валидные SSL-сертификаты

## 🚀 Установка и запуск

1. **Клонируйте репозиторий**
```bash
git clone https://github.com/gambit295/webrtc_p2p.git
cd p2p-learning
```

2. **Создайте SSL сертификаты** (для HTTPS)
```bash
mkdir certs
openssl req -x509 -newkey rsa:4096 -keyout certs/key.pem -out certs/cert.pem -days 365 -nodes
```

3. **Запустите сервер**
```bash
go run main.go
```

4. **Откройте приложение**
```
https://localhost:8443
```

## 🔧 Конфигурация

### Сервер (main.go)
- **Порт**: 8443 (HTTPS)
- **WebSocket эндпоинт**: `/ws`
- **STUN серверы**: Google Public STUN (можно заменить в `ICE_SERVERS`)

### Клиент (index.html)
- Автоматическое определение протокола (wss/ws)
- Адаптивный дизайн
- Поддержка всех современных браузеров

## 🧪 Тестирование

Для тестирования откройте приложение в двух разных браузерах или на двух разных устройствах:
1. В одном браузере нажмите **"Я Учитель"**
2. В другом браузере нажмите **"Я Ученик"**
3. Разрешите доступ к камере и микрофону
4. Наслаждайтесь общением!

## 🔐 Безопасность

- Все соединения идут по HTTPS/WSS
- P2P трафик шифруется (WebRTC DTLS/SRTP)
- Сигнальный сервер не хранит никаких данных
- Сертификаты можно заменить на валидные для продакшена
